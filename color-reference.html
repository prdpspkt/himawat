Quick Usage Guide

Primary Actions: Use Scarlet (#DA251C) for main CTAs like "Get Consultation", "Enroll Now"

Secondary Actions: Use Emerald (#006D32) for alternatives like "Learn More"

Backgrounds: Use White (#FEFEFE) for content, Off-White (#F9FAFB) for sections

Text Hierarchy: Almost Black for headings, Dim Grey Darker for body text

Links: Always use Scarlet (#DA251C) for consistency
